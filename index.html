Detecting Fake reviews on Amazon

Python PySpark Flask NLP BERT 

The aim of this project is to detect fake reviews on Amazon using the review text. The approach combines semi-supervised learning and transformer models to identify fake Amazon reviews. The end product of this project is a web application which will allow the user to predict whether a review is fake or real.
